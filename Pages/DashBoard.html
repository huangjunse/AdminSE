    轻量级图形库（51k），可画6种图形

    雷达图

    圆环图

    使用方法：
    1. 引入js文件：
    <script src="Chart.min.js" type="text/javascript"></script>
    2. 创建 canvas 并定义大小：
    <canvas height="150px" id="Chart1"></canvas>
    3. 创建 context ：
    var ctx = $('#Chart1').get(0).getContext("2d");
    4. 创建 图形：
    new Chart(ctx).Function(ChartData, Options);
    其中：

    方法使用：
    首先，在创建图形的时候，需要先将图形赋予某个变量作为Charjs对象。以Line为例：
    var myLineChart = new Chart(ctx).Line(data);
    方法列表：
    1. .clear()
    清空图形，在动画过程中用得最多的方法，注意的是，clear之后，该对象依然存在，如果指定了响应式参数，则可能会导致重绘。
    myLineChart.clear();    //用法下同，不再重复叙述
    2. .stop()
    在动画过程中停止循环，如果要继续，请调用 .render() 方法
    3. .resize()
    重置画布大小（重绘），响应式参数仅响应浏览器大小变化，如果容器发生变化将不会响应，因此，这个时候你需要手动调用此方法。
    4. .destroy()
    清理Chartjs对象，并且清除所有事件监听。
    5. .toBase64Image()
    返回画布的当前状态的png图像。
    6. .generateLegend()
    根据标签模板返回一段Html字符串。

    更多内容参见：Chart.js 文档
