    滑动条和模态框

    Jquery Slider

    Jquery Slider 是 Jquery UI 中的一部分，Jquery UI是一个包含拖动、放置、缩放排序等操作的高级UI工具合集，其中的 slider 控件也是拥有较好的用户体验的。

    1. 使用方法：

    1.1. 添加css及js：

    <link href="static/plugins/jquery-ui/jquery-ui.min.css" rel="stylesheet" type="text/css" />
    <script src="static/plugins/jquery-ui/jquery-ui.min.js" type="text/javascript"></script>
    1.2. 产生以 div 为基础的 HtmlDom：
    <div id="JquerySlider1"></div>
    1.3. 激活 Jquery slider 控件：
    $('#JquerySlider1').slider();
    特别提醒：
    由于 Jquery Slider 和 Bootstrap Slider 中 slider 方法冲突，Bootstrap Slider会自动将命名空间更换为 bootstrapSlider ，因此，同时加入这两个slider的时候，注意要将 Jquery UI 放在前面。

    2. 颜色主题：
    白色/primary/success/info/warning/danger/default 7种颜色主题。
    HtmlDom 添加 class="slider-primary", 并在js初始化的时候加上 range 参数即可修改主题，不加则使用白色主题

    3. 参数列表：
    参数 默认值 说明
    animatefalse使用动画，一般仅用于配合value/values方法：true/false
    distance0迷之参数
    max100滑动条最大值
    min0滑动条最小值
    step1拖动步长
    orientation'horizontal'水平还是垂直：horizontal/vertical
    rangefalse 是否是范围滑动条：false/true

    value0滑动条当前值
    valuesnullrange=true的时候，这个参数接受一个两个元素的数组，用于指定数组两端的值
    changenull 可以添加值变化的回调函数，如：
    change: function(){
      console.log('change:',$(this).slider('value'));
    }
    slidenull可以添加滑动时的回调函数
    startnull可以添加滑动开始的回调函数
    stopnull可以添加滑动停止的回调函数

    4. 方法列表：

    4.1 value： 仅当 range!=true 时有意义。
    4.1.1 取值： jquerySliderObj.slider('value'); //4
    4.1.2 赋值： jquerySliderObj.slider('value', 5); or $('JuqerySelector').slider('value', 5);
    4.2 values： 仅当 range==true 时有意义。
    4.2.1 取值： jquerySliderObj.slider('values'); //[2,9]
    4.2.2 赋值： jquerySliderObj.slider('values', [3,6]); or $('JuqerySelector').slider('values', [3,6]);

    Bootstrap Slider

    Bootstrap Slider 是 Jquery UI Bootstrap 中的一部分，CSS 4kb，js 21kb，是轻量的 slider UI插件。

    1. 使用方法：

    1.1. 添加css及js：

    <link href="static/plugins/bootstrap-slider/bootstrap-slider.min.css" rel="stylesheet" type="text/css" />
    <script src="static/plugins/bootstrap-slider/bootstrap-slider.js" type="text/javascript"></script>
    1.2. 产生以 input 为基础的 HtmlDom：
    <input id="BootstrapSlider1" type="text"/>
    1.3. 激活 bootstrap slider 控件：
    $('#BootstrapSlider1').bootstrapSlider();
    特别提醒：
    如果同时加入 Juqery UI 和 Bootstrap Slider （注意先后顺序），命名空间才会变成 bootstrapSlider ，否则就是 slider。

    添加 data-slider-id="primary" 即可修改主题，不加则使用白色主题

    3. js赋值方式参数列表：
    参数 默认值 说明
    id''颜色主题，可选值为primary/success/info/warning/danger/purple
    min0滑动条最小值
    max10滑动条最大值
    step1拖动步长
    precision0小数位数（一般是配合步长使用）
    orientation'horizontal'水平还是垂直：horizontal/vertical
    value5滑动条当前值
    rangefalse是否是范围滑动条：false/true
    selection'before'高亮滑动条是在前面还是后面：before/after
    tooltip'show'标签显示方式：show/hide/always
    tooltip_splitfalse是否分裂标签，一般只在范围滑动条的情况下使用：true/false
    handle'round' 把手样式：round/trangle/square/custom
    其中，如果选择 custom 则需要通过 CSS 为custom定义图标：
    .slider-handle.custom {
      /*此处可以添加png图像*/
      background: transparent none;
    }
    .slider-handle.custom::before {
      /*或者通过这种方式添加字体图标*/
      line-height: 20px;
      font-size: 20px;
      content: '\2605'; /*unicode star character*/
      color: #726204;
    }
    reversedfalse反转滑动条（从右往左/下往上显示）：false/true
    enabledtrue滑动条的生效状态：true/false

    4. 方法：
    使用方式：

    $('JquerySeletor').bootstrapSlider('MethodName'[,'Param1'[,'Param2']])
    方法列表：

    名称 示例 说明
    getValue.bootstrapSlider('getValue') 获取当前值，如果是范围滑动条，会得到一个两个元素的数组
    setValue.bootstrapSlider('setValue',10) 设置当前值，如果是范围滑动条，需要一个两个元素的数组
    destroy.bootstrapSlider('destroy') 销毁滑动条，变回input控件
    disable.bootstrapSlider('disable') 置为无效
    enable.bootstrapSlider('enable') 置为有效
    toggle.bootstrapSlider('toggle') 触发有效/无效状态
    isEnabled.bootstrapSlider('isEnabled') 获取生效状态
    getAttribute.bootstrapSlider('getAttribute','id') 获取所有配置/单个配置
    setAttribute.bootstrapSlider('setAttribute','id','info') 设置某个配置，注意，设置完之后需要使用 refresh 方法刷新滑动条
    refresh.bootstrapSlider('setAttribute','id','info').bootstrapSlider('refresh') 刷新滑动条，一般用于 setAttribute 之后
    relayout.bootstrapSlider('relayout') 重新布局——作用暂时未知，应该与隐藏/显示滑动条有关

    Bootstrap Modal

    模态框是经过优化的弹出框

    1. 使用：

    代码

    <button type="button" class="btn btn-primary" data-toggle="modal" data-target="#myModal">
      仅通过html就可以显示模态框
    </button>
    <button type="button" class="btn btn-primary" id="ShowModal">
      通过js显示模态框
    </button>
    <div class="modal fade" id="myModal" tabindex="-1" role="dialog" aria-labelledby="myModalLabel">
      <div class="modal-dialog">
        <div class="modal-content">
          <div class="modal-header">
            <button type="button" class="close" data-dismiss="modal" aria-label="Close"><span aria-hidden="true">×</span></button>
            <h4 class="modal-title">Modal title</h4>
          </div>
          <div class="modal-body">
          </div>
          <div class="modal-footer">
            <button type="button" class="btn btn-default" data-dismiss="modal">关闭</button>
            <button type="button" class="btn btn-primary">确认</button>
          </div>
        </div><!-- /.modal-content -->
      </div><!-- /.modal-dialog -->
    </div><!-- /.modal -->    <script>
    $('#ShowModal').click(function(event) {
      $('#myModal').modal('show');
    });
    </script>

    2. 尺寸

    为 .modal-dialog 类同时添加 .modal-sm/.modal-lg 可以增加小尺寸/大尺寸的模态框

    大尺寸
    小尺寸

    3. 禁止动画效果

    如果你不需要模态框弹出时的动画效果（淡入淡出效果），删掉 .fade 类即可。


    4. 使用栅格系统

    如果要正常使用栅格系统，需要在 .modal-body 底下紧接着添加一个含有 .container-fluid 类的div：

    <div class="modal-body">
      <div class="container-fluid">
        <div class="row">
          <div class="col-md-4">.col-md-4</div>
        </div>
      </div>
    </div>


    5. 方法：

    .modal(options)

    5.1 toggle：手动打开或关闭模态框。
    5.2 show：手动打开模态框。
    5.3 hide：手动隐藏模态框。
    5.4 remote：如果提供的是 URL，将利用 jQuery 的 load 方法从此 URL 地址加载要展示的内容（只加载一次）并插入 .modal-content 内。如果使用的是 data 属性 API，还可以利用 href 属性指定内容来源地址。


    6. 事件：

    $('#myModal').on('hidden.bs.modal', function (e) {
      /* do something...*/
    })

    6.1 show.bs.modal：show 方法调用之后立即触发该事件。如果是通过点击某个作为触发器的元素，则此元素可以通过事件的 relatedTarget 属性进行访问。
    6.2 shown.bs.modal：此事件在模态框已经显示出来（并且同时在 CSS 过渡效果完成）之后被触发。如果是通过点击某个作为触发器的元素，则此元素可以通过事件的 relatedTarget 属性进行访问。
    6.3 hide.bs.modal6.1 hide 方法调用之后立即触发该事件。
    6.4 hidden.bs.modal6.1 此事件在模态框被隐藏（并且同时在 CSS 过渡效果完成）之后被触发。
    6.5 loaded.bs.modal6.1 从远端的数据源加载完数据之后触发该事件。
