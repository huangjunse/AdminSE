    Bootstrap 栅格及排版

    一、栅格系统

    栅格系统用于通过一系列的行（row）与列（column）的组合来创建页面布局，你的内容就可以放入这些创建好的布局中，如此做的话，页面内容能根据浏览器的大小进行不同的布局变化，达到响应式的效果。
    简单来说：
    1. .row 样式（下称row）创建一个行容器， .col-*-N 样式（下称col）创建一个列容器。
    2. row层级并无实际内容，所有内容均应包含在col中。
    3. col中的N从1~12变化，表示将row 进行了12等分，可以跨越多个列指定列宽度。
    4. col中的*可以是xs/sm/md/lg这4个选项，对应的浏览器尺寸为：
    • xs: 超小屏幕 手机 (<768px)
    • sm: 小屏幕 平板 (≥768px)
    • md: 中等屏幕 桌面显示器 (≥992px)
    • lg: 大屏幕 大桌面显示器 (≥1200px)
    5. 可以使用 .col-*-offset-N 来产生偏移量，达到布局效果。
    6. 通过 .col-*-push-N （后移） 和 .col-*-pull-N （前推） 可以进行排序操作。

    二、排版系统

    1. 标题：
    提供h1~h6的标题大小样式，同时标题内，可以含有small标签标记副标题。
    2. 页面主体：
    Bootstrap 将全局 font-size 设置为 14px，line-height 设置为 1.428。这些属性直接赋予 <body> 元素和所有段落元素。另外，<p> （段落）元素还被设置了等于 1/2 行高（即 10px）的底部外边距（margin）。

    三、响应式布局

    .visible-*-N 和 .hidden-* 将可以按照不同浏览器尺寸进行展示，可以有效进行Dom元素显示和隐藏的控制。
    超小屏幕 手机 (<768px) 小屏幕 平板 (≥768px) 中等屏幕 桌面 (≥992px) 大屏幕 桌面 (≥1200px)
    .visible-xs-N 可见 隐藏 隐藏 隐藏
    .visible-sm-N 隐藏 可见 隐藏 隐藏
    .visible-md-N 隐藏 隐藏 可见 隐藏
    .visible-lg-N 隐藏 隐藏 隐藏 可见
    .hidden-xs 隐藏 可见 可见 可见
    .hidden-sm 可见 隐藏 可见 可见
    .hidden-md 可见 可见 隐藏 可见
    .hidden-lg 可见 可见 可见 隐藏

    AdminLTE 布局

    一、导航栏固定

    为 body 标签添加 fixed 样式即可。同时注意，需要调用方法： 
    AdminLTE.layout.fixSidebar();

     

    二、左侧边栏收缩

    为 body 标签添加 sidebar-collapse 样式即可。
    实际上除了能为主体内容腾出空间，这个方法并不好用，有bug存在，建议搭配 左侧边栏鼠标悬停选项 使用。

     

    三、左侧边栏鼠标悬停选项

    开，则调用
    AdminLTE.pushMenu.expandOnHover();
    关，则调用
    AdminLTE.pushMenu.expandOffHover();
