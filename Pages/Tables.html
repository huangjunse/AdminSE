    表格

    Simple table

    这是一个简易的表格样式，具有如下特点：

    1. 轻量：js文件4kb，css文件10kb。
    2. 使用简单：采用js生成方式仅需一行代码（当然，必要的table Dom还是需要的）。
    3. 可以采用非JS方式：所有实现结果都是通过CSS实现，因此可以通过非js生成。


    1. 使用方法：

    1. 添加CSS及JS文件

    <link href="static/plugins/simpletable/simpletable.min.css" rel="stylesheet" type="text/css" />
    <script src="static/plugins/simpletable/simpletable.min.js" type="text/javascript"></script>
    2. 书写 Table Dom
    <table id="SimpleTable">
      <thead>
        <tr><th>...</th></tr>
      </thead>
      <tbody>
        <tr><td>...</td></tr>
        <tr><td>...</td></tr>
      </tbody>
      <tfoot>
        <tr><th>...</th></tr>
      </tfoot>
    </table>
    3. 采用js方式生成：
    $('#SimpleTable').SimpleTable(options);


    2. 非js方式生成：

    2.1 朴素 SimpleTable ： 为 table 添加 .simpletable 类即可。

    2.2 激活状态： 为 tr 添加 .tr-active 类即可产生改行的激活状态。

    2.3 禁用状态： 为 tr 添加 .tr-disable 类即可产生改行的禁用状态。

    2.4 6种主题： 为 .simpletable 类添加 .tab-* 即可产生5种色彩主题（*可以为primary/success/info/warning/danger）。

    2.5 无边框样式：为 .simpletable 类添加 .noborder 类可以产生无边框的表格效果。

    2.6 极其简单的Dom： table 中可以仅仅含有 tr th td 等元素而不必含有 thead tbody tfoot 等元素（尽管推荐使用这些标签）。

    2.7 对齐状态：为 .simpletable 类添加 .text-* 即可不同的对齐状态（*可以为left/center/right）。

    2.8 响应式表格：为表格外层添加一个div，并附上 .table-responsive 类，可以达到响应式的效果（依赖bootstrap）。


    表头1表头2表头3
    这是一行普通行
    这是一行禁用状态的行
    这是一行激活状态的行

    <table class="simpletable tab-primary">
      <tr class=""><th>表头1</th><th>表头2</th><th>表头3</th></tr>
      <tr><td>这是</td><td>一行</td><td>普通行</td></tr>
      <tr class="tr-disable"><td>这是</td><td>一行</td><td>禁用状态的行</td></tr>
      <tr class="tr-active"><td>这是</td><td>一行</td><td>激活状态的行</td></tr>
    </table>

    4. 参数(options)：

    参数默认值描述
    hoverfalse是否产生鼠标经过效果：false/true
    paddingfalse是否产生紧缩的表格样式：false/'no'/'tiny'/'tight'
    bordertrue是否有边框：true/false
    color''颜色主题：''/'primary'/'success'/'info'/'warning'/'danger'
    algin'center'对齐状态：'left'/'center'/'right'
    参数默认值描述

    3. js方法：采用纯粹 Html 方式虽然能展现出效果不错的表格，但是一些需要js支持的功能却无法使用，这是一种遗憾。
    因此在插件中添加几个较为有用的方法，提供使用：

    方法示例描述
    addHover 
    .SimpleTable('addHover');
    		 为表格添加鼠标经过样式
    addColumn 
    .SimpleTable('addColumn',options);
    		 批量为表添加列，参数options介绍（以下为默认值）：
    {
      colCount:1, //需要添加的列数
      contents: [], //嵌套数组，用于存放需要添加的内容
      hasID: false, //是否需要为每个单元格添加id属性
      headIDPrefix: 'H', //th的id前缀
      lineIDPrefix: 'L', //td的id前缀
      startID: 0 //id的起始序号
    }
    rowspan 
    .SimpleTable('rowspan','all');.SimpleTable('rowspan',0);
    		 合并内容相同的单元格为跨行单元格：'all'/行号(起始为0)
    colspan 
    .SimpleTable('colspan','all');.SimpleTable('colspan',0);
    		 合并内容相同的单元格为跨行单元格：'all'/列号(起始为0)
    rowactive 
    .SimpleTable('rowactive','all');.SimpleTable('rowactive',0);
    		 将全部行/制定行置为禁用状态：'all'/行号(起始为0)
    注意，对标题行不能生效。
    rowdisable 
    .SimpleTable('rowdisable','all');.SimpleTable('rowdisable',0);
    		 将全部行/制定行置为禁用状态：'all'/行号(起始为0)

    Code: 
    $('#SimpleTable3').SimpleTable('rowspan','all');
    $('#SimpleTable3').SimpleTable('colspan','all');

    特别需要注意以下内容：
    1. 由于“被合并”的单元格并非实际删除，而仅仅只是隐藏了（加上了display:none;），因此每一行的数据依然还是存在的，在进行单元格操作的时候注意别忘了这个单元格。
    2. 一般情况下只会存在跨行或者跨列的情况，不会存在既跨行又跨列的情况，这个在Html中是没有办法通过简单方式实现的，如果存在跨越多行及多列的情况，切记一定要保证所有跨越的行和列都是同一个值（如例子中的TD4.3），否则将产生不可预知的错位。
    3. th 和 td 之间不能跨越，所以请保证数据首行（或者尾行）与 标题行没有相同的单元格。

    DataTable

    Datatables是一款jquery表格插件。它是一个高度灵活的工具，可以将任何HTML表格添加高级的交互功能。

    • 分页，即时搜索和排序
    • 几乎支持任何数据源：DOM， javascript， Ajax 和 服务器处理
    • 支持不同主题 DataTables, jQuery UI, Bootstrap, Foundation
    • 各式各样的扩展: Editor, TableTools, FixedColumns ……
    • 丰富多样的option和强大的API
    • 支持国际化
    • 超过2900+个单元测试
    • 免费开源 （ MIT license ）！ 商业支持
    • 更多特性请到官网查看

    【特别备注】
    当前集成到AdminSE中的DataTables是经过了本地化的插件，已经将常见的几个英文语句汉化了，而且不可逆。
    如果不喜欢定制化的东西，请上官网去下载。

    1. 使用：
    1.1 CSS及JS：

    <link href="static/plugins/datatables/dataTables.bootstrap.css" rel="stylesheet" type="text/css"/>
    <script src="static/plugins/datatables/jquery.dataTables.min.js" type="text/javascript"></script>
    2. 书写 Table Dom
    同SimpleTable，不再赘述。
    3. 初始化DataTable 
    $('#DataTable').DataTable(options);

    2. 参数(options)：

    这个工具是强大的，所有参数非常之多，这里不进行详细描述，仅列举常用的几个。
    有需要去中文网看选项：DataTable选项

    参数默认值描述
    autoWidthtrue控制Datatables是否自适应宽度：true/false
    infotrue控制是否显示表格左下角的信息：true/false
    jQueryUIfalse控制是否使用jquerui的样式（需要引入jqueryui的css）：true/false
    lengthChangetrue是否允许用户改变表格每页显示的记录数：true/false
    orderingtrue是否允许Datatables开启排序：true/false
    pagingtrue是否开启本地分页：true/false
    scrollXfalse设置水平滚动：true/false
    scrollYfalse设置垂直滚动：true/false
    searchingtrue是否允许Datatables开启本地搜索：true/false
    stateSavefalse保存状态 - 在页面重新加载的时候恢复状态（页码等内容）：true/false

    【彩蛋】：DataTable注重更多的是功能而非样式，而SimpleTable注重的是样式多过功能，两者可以搭配起来使用：

    TD1TD2TD3
    TD1TD2TD3
