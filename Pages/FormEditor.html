    富文本编辑器

    bootstrap wysiwyg

    这是一个优秀的轻量富文本编辑器！

    你可以进行大量的定制化操作！

    此版本是经过本人定制过的，具有如下特性：

    1. 自动生成工具栏，工具栏含有23个常见的工具。
    2. 工具栏的功能可定制。
    3. 工具栏中的字体列表可以定制。
    4. 修复源代码中快捷键不能使用的bug。
    5. 上传图片功能采用回调函数的方式，灵活性增加
    6. 版本号定义为wysiwyg V1.0sam


    一、使用方法：

    1. 添加css及js文件：

    /*可选属性：定义编辑器高度，不定义的话编辑器只有一行*/
    #wysiwygEditor {overflow:scroll; max-height:300px;height:300px}
    <link href="static/plugins/bootstrap-wysiwyg/bootstrap-wysiwyg.css" rel="stylesheet" type="text/css" />
    <script src="static/plugins/bootstrap-wysiwyg/bootstrap-wysiwyg.js" type="text/javascript"></script>

    2. 添加HtmlDom：

    <div id="wysiwygEditor"></div>

    3. js激活：

    $('#wysiwygEditor').wysiwyg(options);

    二、获取和写入内容：

    由于直接采用的是HtmlDom方式产生内容，因此，直接使用jquery.html()函数即可获取和写入内容。

      


    三、参数(options)：

    参数 默认值 描述
    toolbarSelector '[name="wysiwygEditor"]' 工具栏的选择器。
    由于定制化后的wysiwyg的工具栏是自动生成的，
    采用了默认的name，因此，建议不要更改此选项。
    commandRole 'edit' 工具栏产生命令的data属性。
    由于定制化后的wysiwyg的工具栏是自动生成的，
    采用了默认的edit，因此，建议不要更改此选项。
    toolbarItems 
    ['undo','redo','font','fontsize','frontcolor','backcolor','bold','italic','strikethrough','underline','unorderedlist','orderedlist','outdent','indent','alignleft','aligncenter','alignright','alignjustify','addlink','unlink','insertimage','unformat','source'],
    		 可以定制需要哪些工具栏，哪些不要。
    activeToolbarClass 'btn-info' 选中的工具高亮主题，
    可选为 btn-default/btn-primary/
    btn-success/btn-info/
    btn-warning/btn-danger
    fontItems 
    ['宋体', '幼圆', '黑体', '楷体', '隶书',  '华文彩云', '华文细黑', '华文行楷','Sans', 'Arial', 'Arial Black', 'Courier New', 'Comic Sans MS', 'Helvetica', 'Tahoma', 'Times New Roman', 'Verdana']
    		 可选的字体，完全可以自由定制，
    只要是常见的系统自带字体即可。
    hotKeys 
    {
        'ctrl+b': 'bold',
        'ctrl+i': 'italic',
        'ctrl+u': 'underline',
        'ctrl+z': 'undo',
        'ctrl+y': 'redo',
        'ctrl+l': 'justifyleft',
        'ctrl+r': 'justifyright',
        'ctrl+e': 'justifycenter',
        'ctrl+j': 'justifyfull',
        'shift+tab': 'outdent',
        'tab': 'indent'
    }
    		 定义快捷键，目前仅能从上面几个中选择。
    insertImage null 点击“插入图片”按钮的回调函数，
    由于上传功能需要搭配后台，因此分离该功能，示例：
    insertImage: function(){
      console.log('click insertImage button')
    }
    dragAndDropImages false 此为定制前的工具栏上传图片功能，
    目前已经弃用该功能，因此该参数不可用。
    selectionMarker 'edit-focus-marker' 谜之参数，建议不要更改
    selectionColor 'darkgrey' 谜之参数，建议不要更改
